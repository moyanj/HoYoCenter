用户数据和隐私权益

个人信息数据：

HoYoGameLauncher 的项目性质决定了其不会收集任何个人信息数据

常规数据使用：

HoYoGameLauncher 充分尊重用户的隐私，目前我们会按下方的说明使用用户的数据：

  • 使用HoYoGameLauncher 后，在 <安装目录>/data/player 中以二进制文件的形式

    • 储存
      • 米游社和 HoYoLab Cookie
      • 玩家昵称
      • 玩家UID

  • 使用 HoYoGameLauncher 后，在 <安装目录> 中以JSON文本文件的形式

    • 储存
      • 程序配置文件

  • 使用 HoYoGameLauncher 主程序时， HoYoGameLauncher 会因开发和统计为目的匿名化地收集程序意外停止或出错时抛出的日志，包括：

    • 错误与崩溃日志
    • 设备的类型；
    • 操作系统版本；
    • 国家，语言；
    • 内存信息；

  • 使用 胡桃数据库 (深渊记录/深渊统计) 功能并选择上传数据时，

    • 向 胡桃 API 上传
      • 用户的游戏 UID
      • 用户的深境螺旋记录
      • 用户的角色信息及其装备的武器和圣遗物信息

  • 使用MoYanACT功能后，MoYanACT云端系统会

    • 储存
      • 注册使用的邮箱
      • 加密后的密码
