HoYoGameLauncher

       

此存储库包含了HoYoGameLauncher的源代码。HoYoGameLauncher是一个米哈游游戏启动器，旨在简化玩家启动游戏的步骤。它功能强大，维护积极，易于使用。

介绍  •  安装  •  开发文档

# 介绍

HoYoGameLauncher提供了一个米哈游聚合启动器，可以快捷的启动：

  • 崩坏：星穹铁道
  • 原神
  • 原神国际服（暂未实现）
  • 崩坏：星穹铁道国际服（暂未实现）
  • 崩坏3（暂未实现）

# 安装

下载最新releases版本open in new window
下载最新开发版本open in new window(可能无法正常运行)

  • 如果你下载了exe安装包, 请直接运行，然后根据提示安装。
  • 如果你下载了zip程序压缩包, 请解压后双击打开HoYoGameLauncher.exe即可运行。（注意：您可能需要安Microsoft Edge WebView2）
  • 如果你下载了源代码, 请前往开发文档open in new window查看如何运行。
